# Container Components #
Groups nested HVY components.
Use children-order.json to reorder container children.
containerBlocks holds the child block array.
containerTitle is an optional reader label for the collapse/expand control.
containerExpanded stores the default expanded state.
containerCollapsedPreviewRem controls the collapsed preview height in rem units and defaults to 3.
Plain Markdown before the first numbered container slot is treated as the first implicit child block.
Numbered container slots order children by their numeric suffix; slot markers carry only slot metadata.
